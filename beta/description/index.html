Beta Description

Handshake post feature: see it in feed on Handshake!

Must be logged in to view (free account), this description is specifically for a longer post needing the "read more" feature.

TRIGGERS

Handshake Post Container

Read More

  1. Hover On "...More" button
  2. Hover Off "...More" button
  3. Click "...More" button

Like Button

  1. Hover On the heart Icon
  2. Hover Off the heart Icon
  3. Click the heart Icon
  4. Hover On the number of likes
  5. Hover Off the number of likes
  6. Click to reveal who has liked

Bookmark Button

  1. Hover On the bookmark Icon
  2. Hover Off the bookmark Icon
  3. Click the bookmark Icon

Comment Button

  1. Hover On the speech bubble Icon
  2. Hover Off the speech bubble Icon
  3. Click the speech bubble Icon

Post Contents

  1. Hover On links in the post
  2. Hover Off the links in the post
  3. Click Post

Rules

Read More

Hover On

The "...More" button will become underlined on hover

Hover Off

The "...More" button will return to its original state, non-underlined

Click

The caption will expand on the top of the post, image connected with post will shift downward giving more room for the now elongated caption

Like Button

Hover On

Gray circle circle appears behind the heart icon

The number becomes underlined

Hover Off

Gray circle disappears from the heart icon

The underline disappears from the number of likes

Click

Heart icon turns into pink
The number of likes will reveal what users have liked the post

Bookmark Button

Hover On

Gray circle circle appears behind the bookmark icon

Hover Off

Gray circle disappears from the bookmark icon

Click

Bookmark icon turns into purple

Comment Button

Hover On

Gray circle circle appears behind the speech bubble icon

Hover Off

Gray circle disappears from the speech bubble icon

Click

Opens the full post allowing user to comment

Post Content

Hover On

Links in the post will become underlined

Hover Off

Underlined links will return to original state

Click

Opens the full post
