Beta Description

Handshake post feature: see it in feed on Handshake!

Must be logged in to view (free account), this description is specifically for a longer post needing the "read more" feature.

TRIGGERS

Handshake Post Container

More Button

  1. Hover On "...More"
  2. Hover Off "...More"
  3. Click "...More"

Heart Button

  1. Hover On the heart Icon
  2. Hover Off the heart Icon
  3. Click the heart Icon

Bookmark Button

  1. Hover On the bookmark Icon
  2. Hover Off the bookmark Icon
  3. Click the bookmark Icon

Post Contents

  1. Hover On the content in the post
  2. Hover Off the content in the post

Rules

More Button

Hover On

  1. The "...More" button will swiftly change color on hover
  2. Cursor becomes pointer

Hover Off

  1. The "...More" button will return to its original state
  2. Pointer returns to cursor

Click

  1. The caption will expand on the top of the post, image connected with post will shift downward giving more room for the now elongated caption

Heart Button

Hover On

  1. Heart icon fills in with a darker color
  2. Cursor becomes pointer

Hover Off

  1. The heart icon returns to its original state
  2. Pointer returns to cursor

Click

  1. Heart icon turns into pink

Bookmark Button

Hover On

  1. The bookmark icon fills in with a darker color
  2. Cursor becomes pointer

Hover Off

  1. The bookmark icon fills in with a darker color
  2. Cursor becomes pointer

Hover On

  1. The bookmark icon returns to its original state
  2. Pointer returns to cursor

Click

  1. Bookmark icon turns into purple

Post Content

Hover On

  1. Link becomes emphasized
  2. Cursor becomes pointer

Hover Off

  1. Image returns to its original state
  2. Pointer returns to cursor
