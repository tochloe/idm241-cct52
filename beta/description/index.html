Beta Description

Handshake post feature: see it in feed on Handshake!

Must be logged in to view (free account), this description is specifically for a longer post needing the "read more" feature.

TRIGGERS

Handshake Post Container

Read More

  1. Hover On "...More" button
  2. Hover Off "...More" button
  3. Click "...More" button

Like Button

  1. Hover On the heart Icon
  2. Hover Off the heart Icon
  3. Click the heart Icon

Bookmark Button

  1. Hover On the bookmark Icon
  2. Hover Off the bookmark Icon
  3. Click the bookmark Icon

Post Contents

  1. Hover On the content in the post
  2. Hover Off the content in the post

Rules

Read More

Hover On

The "...More" button will swiftly change color on hover

Cursor becomes pointer

Hover Off

The "...More" button will return to its original state

Pointer returns to cursor

Click

The caption will expand on the top of the post, image connected with post will shift downward giving more room for the now elongated caption

Like Button

Hover On

Cursor becomes pointer

Hover Off

The heart icon returns to its original state

Pointer returns to cursor

Click
